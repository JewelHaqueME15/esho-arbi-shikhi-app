# Esho Arbi Shikhi (এসো আরবি শিখি)

An interactive Arabic learning web app. The entire app is a single HTML file:
[esho-arbi-shikhi-app.html](esho-arbi-shikhi-app.html).

## Deployment

Deployed on Vercel. `vercel.json` rewrites the root URL `/` to the app file,
so the app opens directly at the site root. Every push to `main` triggers an
automatic redeploy.

## Editing

Edit `esho-arbi-shikhi-app.html`, then commit and push:

```
git add -A
git commit -m "Update app"
git push
```
